# 14) Toast + progress patterns

লেম্যান-বাংলা: লং রানিং একশন দেখাতে toast + progress বার, শেষে success/error।

## Hands-on
1) চালান:
   ```bash
   cd performance-and-ux/demos/toast-progress-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে toast state কিভাবে info→success হয় দেখুন।
3) error অবস্থা যোগ করে rerun করুন।

## Done when…
