# 12) Lighthouse + bundle budgets

লেম্যান-বাংলা: bundle বাজেট সেট করে CI/Lighthouse রিপোর্ট চেক করুন।

## Hands-on
1) চালান:
   ```bash
   cd performance-and-ux/demos/lighthouse-budgets-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে JS/CSS/IMG বাজেট পাস/ফেইল দেখুন।
3) বাজেট বা রিপোর্ট মান বদলে ফেইল/পাস সিমুলেট করুন।

## Done when…
- বাজেট লেখা ও চেক করা হয়; ফেইলে পাইপলাইন লাল হয়।
