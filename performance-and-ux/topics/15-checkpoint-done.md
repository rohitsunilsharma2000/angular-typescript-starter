# 15) Checkpoint: Definition of Done (performance)

লেম্যান-বাংলা: রিলিজের আগে পারফ-চেকলিস্ট।

## Hands-on
1) চালান:
   ```bash
   cd performance-and-ux/demos/perf-checkpoint-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে পারফ DoD তালিকা দেখুন; আপনার টিমের জন্য আইটেম যোগ/কমান।

## Done when…
- টিম-সম্মত পারফ চেকলিস্ট আছে এবং CI/রিভিউতে মিলিয়ে দেখা হয়।
