# 11) Profiling: Angular DevTools + Chrome

লেম্যান-বাংলা: DevTools/Performance রেকর্ড করে long task ও CD সাইকেল দেখুন।

## Hands-on
1) চালান:
   ```bash
   cd performance-and-ux/demos/profiling-devtools-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে প্রোফাইলিং চেকলিস্ট দেখুন; নিজের স্টেপ যোগ করুন।
3) বাস্তবে Chrome Performance + Angular DevTools এ একই স্টেপ অনুসরণ করুন।

## Done when…
- long task শনাক্তের স্টেপ জানেন; রেকর্ডিং করতে পারেন।
