# 10) Virtual scroll + trackBy

লেম্যান-বাংলা: বড় তালিকায় virtual scroll + trackBy/id দিয়ে DOM স্টেবল রাখুন।

## Hands-on
1) চালান:
   ```bash
   cd performance-and-ux/demos/virtual-scroll-trackby-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে reorder হলেও id তালিকা একই—trackBy আইডিয়া দেখুন।
3) ডুপ্লিকেট id দিলে কেন সমস্যা হবে ভাবুন।

## Done when…
- বড় তালিকায় trackBy সেট; virtual scroll বিবেচনা।
