# 08) SSR/SSG + hydration

লেম্যান-বাংলা: সার্ভার HTML আগে পাঠান, ক্লায়েন্টে hydrate করে ইভেন্ট জুড়ুন।

## Hands-on
1) চালান:
   ```bash
   cd performance-and-ux/demos/ssr-ssg-hydration-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে SSR HTML ও Hydration লগ দেখুন।
3) HTML বদলিয়ে rerun; ভাবুন hydrate না হলে ইভেন্ট কাজ করবে না।

## Done when…
- SSR/SSG আউটপুট ও hydrate step বোঝেন; hydration errors এড়ান।
