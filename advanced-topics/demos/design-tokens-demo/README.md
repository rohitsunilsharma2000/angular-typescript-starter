# Design tokens demo

Simple console demo showing tokens and how they compose into button/card styles.

## Commands
```bash
npm install
npm run demo
npm run typecheck
```

## Expected output
- Printed token JSON
- Button style string with primary/onPrimary colors
- Card style string with surface/text colors

## Try
- Change `primary` color and rerun
- Adjust `spacing` multiplier to see padding change
