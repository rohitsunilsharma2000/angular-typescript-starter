# 09) Integration tests (feature route level)

লেম্যান-বাংলা: ফিচার রুট ধরে happy path টেস্ট করুন (mock API সহ)।

## Hands-on
1) চালান:
   ```bash
   cd testing-and-quality/demos/integration-tests-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে patients route flow ধারণা দেখুন; আপনার ফিচার অনুযায়ী ধাপ লিখুন।

## Done when…
- রুট-লেভেল happy path টেস্ট, mock API সহ, নেভিগেশন চেক।
