# 15) Commit hooks + CI quality gates

লেম্যান-বাংলা: pre-commit/push এ লিন্ট/টেস্ট; CI তে সম্পূর্ণ গেট।

## Hands-on
1) চালান:
   ```bash
   cd testing-and-quality/demos/ci-hooks-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে হুক/CI স্টেপ দেখুন; আপনার টিমের কমান্ড বসান।

## Done when…
- হুক/CI তালিকা স্পষ্ট; রিপোতে কনফিগার করা।
