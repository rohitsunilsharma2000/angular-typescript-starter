# 02) Test runner setup: Jest বা Vitest

লেম্যান-বাংলা: একটি রানার বেছে নিন (Jest/Vitest) এবং jsdom/env ঠিক করুন।

## Hands-on
1) চালান:
   ```bash
   cd testing-and-quality/demos/jest-vitest-setup-demo
   npm install
   npm run demo
   ```
2) আউটপুটে Jest/Vitest config হিন্ট দেখুন।
3) আপনার প্রজেক্টে বাস্তব config (preset/env) বসান।

## Done when…
- টেস্ট রানার ঠিকমতো jsdom/ts সহ চলে; npm test সবুজ।
