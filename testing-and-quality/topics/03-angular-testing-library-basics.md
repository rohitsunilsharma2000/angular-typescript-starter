# 03) Angular Testing Library basics

লেম্যান-বাংলা: Arrange-Act-Assert; কীবোর্ড/রোল দিয়ে কুয়েরি করুন।

## Hands-on
1) চালান:
   ```bash
   cd testing-and-quality/demos/atl-basics-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে AAA ফ্লো দেখুন; নিজের টেস্টে একই ধাপ অনুসরণ করুন।

## Done when…
- role/label কুয়েরি ব্যবহার; user-event দিয়ে act; expect দিয়ে assert।
