# 04) TestBed fundamentals (standalone friendly)

লেম্যান-বাংলা: Standalone component হলে `imports` এ সেটি দিন; detectChanges চালান।

## Hands-on
1) চালান:
   ```bash
   cd testing-and-quality/demos/testbed-standalone-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে TestBed.configureTestingModule + createComponent ধাপ দেখুন।
3) আপনার কম্পোনেন্ট/প্রোভাইডার নাম বসিয়ে চিন্তা করুন।

## Done when…
- Standalone কম্পোনেন্ট টেস্ট করতে TestBed imports কীভাবে লিখবেন জানেন।
