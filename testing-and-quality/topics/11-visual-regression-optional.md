# 11) Visual regression (optional)

লেম্যান-বাংলা: স্ক্রিনশট ডিফে UI পরিবর্তন ধরুন; ছোট প্রজেক্টে optional, ডিজাইন-হেভি হলে দরকারি।

## Hands-on (concept demo)
1) চালান:
   ```bash
   cd testing-and-quality/demos/visual-regression-demo
   npm install
   npm run demo
   npm run typecheck
   ```
2) আউটপুটে স্ক্রিনশট ডিফ ধারণা দেখুন; আপনার টুল (Percy/Chromatic/Playwright) বেছে নিন।

## Done when…
- গুরুত্বপূর্ণ স্ক্রিনে বেসলাইন আছে; ডিফ হলে CI লাল।
